Virtual Assistant

Virtual Assistant


Project Description

My Dad is getting older, is mostly home-bound, and is starting to have trouble with his memory. I want to give him something to do because he doesn't have much that takes up his time. He's interested in history, especially that of the Civil War, and I think he'd enjoy listening to podcasts.

This virtual assistant uses Beautiful Soup to scrape the web for podcasts and Pygame's mixer module to play them. The first website I'm working with is Fred Kiger's Threads From the National Tapestry: Stories From the American Civil War.

My Virtual Assistant repository is stored on GitHub.

Wordle

Wordle


Project Description

I started coding Wordle at 2:30 in the morning. After my roommate introduced me to the game, I started thinking about how I’d develop it and I couldn’t stop. In Udemy’s 100 Days of Python course, I had been working with Turtle but had not yet been introduced to tkinter. Therefore my first iteration used Turtle but I needed clickable buttons to represent the keyboard’s keys and so I switched the GUI to tkinter.

My version of Wordle uses the Datamuse API to generate and validate words.

My Wordle repository is stored on GitHub.

Timers

Timers


Project Description

Because I tend to cook for the week, I’m often working from multiple recipes at the same time. The kitchen has two timers, an oven timer that’s difficult to use and a microwave timer. I needed more timers. Programming up this Timers application in Python and tkinter made cooking much easier.

In Udemy’s 100 Days of Python course, we had not yet learned how to implement modal dialogs. I developed my code with the help of Code Maven's custom simple dialog and Python's simpledialog documentation.

My Timers repository is on GitHub.

Stock News Alert

Stock News Alert


Project Description

This application sends out a text message when Tesla’s closing stock value changes by more than five percent over two consecutive days. This project was developed as a class exercise for Dr. Angela Yu’s 100 Days of Python course. She provided a general description of what the application should do as well as links to Twilio for SMS messaging, the News API for articles related to the stock, and the Alpha Vantage API for stock data.

My Stock News Alert repository is hosted on GitHub.
